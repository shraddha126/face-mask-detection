# Face Mask Detection: FastAPI + Docker

This project implements a Face Mask Detection system using TensorFlow Lite
with a FastAPI backend and Dockerized deployment.

## Features
- FP32 and INT8 TFLite inference
- FastAPI REST API
- Dockerized server
- Local webcam & image clients
- Modular project structure
- Real-time mask detection using webcam.
- Trained on custom dataset.
- Outputs bounding boxes with labels: "Mask" / "No Mask".
